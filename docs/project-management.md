#  Gestión del Proyecto - SafePets
Para el desarrollo de la aplicación Fullstack **SafePets**, hemos adoptado un enfoque ágil híbrido basado en **Kanban**, utilizando la herramienta digital **Trello** para monitorizar el progreso de las tareas en tiempo real de forma visual.

## 1. Estructura del Tablero Kanban
El flujo de trabajo se divide en cinco estados dinámicos para evitar cuellos de botella:

* **Backlog:** Lista maestra con todos los requisitos e historias de usuario de la aplicación.
* **Todo (Por hacer):** Tareas seleccionadas prioritarias listas para ser iniciadas en el ciclo actual.
* **In Progress (En proceso):** Tareas que se están desarrollando activamente en este momento.
* **Review (En revisión):** Código terminado pendiente de pruebas lógicas, verificación de tipos e inspección del linter.
* **Done (Terminado):** Funcionalidades completamente estables, tipadas y desplegadas.
